Atsu1209 PushKeyLib

LibName/apl-1209

PushKeyLib
キーが押されたのを検知するJavaScriptライブラリで,HTMLの指定したオブジェクトをクリックします。

File
1.NumberLib
2.BasicLib
3.BigBasiclib
4.AllLib

1.NumberLib
数字キーが押されたのを検知する

2.BasicLib
小文字のキーを検知

3.BigBasicLib
大文字を検知(Shift+キー)

4.AllLib
数字,小文字,大文字すべてを検知

Exp
例えば数字だけを使う場合動作を軽くするため、NumberLibを使用してください。

HowToUse
Scriptタグを書き,srcタグを使用して1~4のlibを指定してください。
次に classで、 button+指定したいキー を書けばクリック対象となります。

Attention
コピー・複製や再配布は絶対におやめください。
button1 などの番号のクラスをつけると、クリック対象になるので、IDをご使用ください
